# eHrvatska Akademija – Strategija
eRepublik - Strategija, vizija i analize za eHrvatsku i RL Hrvatsku

Ovaj repozitorij sadrži strateški okvir za:
- eHrvatsku u eRepubliku
- RL Hrvatsku (na razini vizije, strategije i modela upravljanja)

Fokus:
- Vizija, misija, vrijednosti
- Strateški modeli i alati (SWOT, PESTLE, OKR, KPI)
- Državna, vojna, diplomatska i društvena strategija
- Presjek znanja primjenjiv i u igri i u stvarnosti

Struktura:
- `0-Temelj/` – osnovni koncepti i svrha
- `1-Vizija-Misija-Vrijednosti/` – identitet i smjer
- `2-Modeli-Planiranja/` – alati za strateško planiranje
- `3-Drzavna-Strategija/` – specifični dokumenti za eHR i RL HR
- `4-Diplomacija-i-Medunarodni-Odnosi/` – vanjski odnosi i savezi

Repozitorij je zamišljen kao temelj za GitBook dokumentaciju (Akademija Strategije).
